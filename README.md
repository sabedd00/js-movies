# Installation

<table>
 <th> <b>git clone</b> https://github.com/sabedd00/js-movies.git</th>
</table>

After creation, your project should look like this:

```
js-movies/
  src/
    css/
      main.css
      movie-details.css
    docs/
      images/
        main-page.png
        movie-details.png
        slider-of-similar-movie.gif
    images/
      empty-poster.png
      tmdb-logo.svg
    js/
      config.js
      index.js
      movie-details.js
      search-results.js
  .gitignore    
  README.md
  index.html    
  package.json
  package-lock.json
  webpack.config.js
```

In the project directory, you can run:

### `npm run build`
### `npm start`
